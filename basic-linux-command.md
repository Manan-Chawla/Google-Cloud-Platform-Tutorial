# Linux Command Line Master Reference

A comprehensive guide to 60 essential Linux commands for file management, system administration, and networking.



## 1. File & Directory Navigation
* **`ls`**: Lists directory contents.
    ```bash
    ls -lah
    ```
* **`pwd`**: Prints the current working directory path.
    ```bash
    pwd
    ```
* **`cd`**: Changes the current directory.
    ```bash
    cd /home/user/Documents
    ```
* **`mkdir`**: Creates a new directory.
    ```bash
    mkdir project_folder
    ```
* **`rmdir`**: Removes an empty directory.
    ```bash
    rmdir old_folder
    ```
* **`ln`**: Creates links between files (use `-s` for symbolic/soft links).
    ```bash
    ln -s target_file link_name
    ```

## 2. File Operations
* **`rm`**: Deletes files or directories (`-r` for recursive).
    ```bash
    rm -rf temporary_file.txt
    ```
* **`cp`**: Copies files or directories.
    ```bash
    cp source.txt destination.txt
    ```
* **`mv`**: Moves or renames files or directories.
    ```bash
    mv old_name.txt new_name.txt
    ```
* **`touch`**: Creates a new empty file.
    ```bash
    touch notes.txt
    ```
* **`file`**: Determines the file type.
    ```bash
    file archive.tar.gz
    ```
* **`locate`**: Finds files by name quickly using a background database.
    ```bash
    locate config.php
    ```
* **`find`**: Searches for files in a directory hierarchy based on various criteria.
    ```bash
    find . -name "*.log"
    ```

## 3. Archiving & Compression
* **`zip` / `unzip`**: Compresses or extracts ZIP archives.
    ```bash
    zip docs.zip *.txt
    unzip docs.zip
    ```
* **`tar`**: Bundles files into a tarball (archive).
    ```bash
    tar -cvf backup.tar /home/user/data
    ```

## 4. Text Editing & Viewing
* **`nano` / `vi` / `jed`**: Command-line text editors.
    ```bash
    nano configuration.conf
    ```
* **`cat`**: Displays the entire content of a file.
    ```bash
    cat version.txt
    ```
* **`head`**: Prints the first few lines of a file (default 10).
    ```bash
    head -n 5 log.txt
    ```
* **`tail`**: Prints the last few lines of a file (use `-f` to follow).
    ```bash
    tail -f access.log
    ```
* **`echo`**: Prints text or variables to the shell.
    ```bash
    echo "Path is: $PATH"
    ```

## 5. Text Processing & Searching
* **`grep`**: Searches for specific patterns within text or files.
    ```bash
    grep "ERROR" system.log
    ```
* **`sed`**: Stream editor for filtering and transforming text.
    ```bash
    sed 's/old-text/new-text/g' file.txt
    ```
* **`awk`**: Pattern scanning and processing language.
    ```bash
    awk '{print $1}' data.csv
    ```
* **`sort`**: Sorts lines of text files.
    ```bash
    sort -n numbers.txt
    ```
* **`cut`**: Extracts specific sections/columns from each line.
    ```bash
    cut -d',' -f2 data.csv
    ```
* **`diff`**: Compares two files line by line.
    ```bash
    diff file1.txt file2.txt
    ```
* **`tee`**: Reads from input and writes to both standard output and files.
    ```bash
    ls | tee output.txt
    ```

## 6. Permissions & User Management


* **`sudo`**: Executes a command with administrative/root privileges.
    ```bash
    sudo apt update
    ```
* **`su`**: Switches to another user account (usually root).
    ```bash
    su - username
    ```
* **`whoami`**: Displays the current logged-in username.
    ```bash
    whoami
    ```
* **`chmod`**: Changes file or directory access permissions.
    ```bash
    chmod 755 script.sh
    ```
* **`chown`**: Changes file or directory ownership.
    ```bash
    sudo chown user:group file.txt
    ```
* **`useradd` / `passwd` / `userdel`**: Creates, sets passwords for, and deletes users.
    ```bash
    sudo useradd newuser
    sudo passwd newuser
    ```

## 7. System Monitoring & Performance
* **`df`**: Reports file system disk space usage.
    ```bash
    df -h
    ```
* **`du`**: Estimates file space usage for a directory.
    ```bash
    du -sh /var/log
    ```
* **`top` / `htop`**: Displays real-time system processes and resource consumption.
    ```bash
    htop
    ```
* **`ps`**: Reports a snapshot of the current processes.
    ```bash
    ps aux
    ```
* **`uname`**: Prints system information (use `-a` for all).
    ```bash
    uname -a
    ```
* **`hostname`**: Shows or sets the system's host name.
    ```bash
    hostname
    ```
* **`time`**: Measures how long a command takes to execute.
    ```bash
    time ./backup_script.sh
    ```
* **`systemctl`**: Controls the systemd system and service manager.
    ```bash
    sudo systemctl restart nginx
    ```
* **`watch`**: Executes a program periodically, showing output in full screen.
    ```bash
    watch -n 1 date
    ```
* **`jobs`**: Lists the active jobs in the current shell.
    ```bash
    jobs
    ```
* **`kill`**: Sends a signal to a process (usually to terminate it).
    ```bash
    kill -9 1234
    ```
* **`shutdown`**: Powers off or reboots the system.
    ```bash
    sudo shutdown now
    ```

## 8. Networking
* **`ping`**: Sends ICMP ECHO_REQUEST to network hosts to check connectivity.
    ```bash
    ping google.com
    ```
* **`wget`**: Non-interactive network downloader.
    ```bash
    wget [https://example.com/file.iso](https://example.com/file.iso)
    ```
* **`curl`**: Transfers data to or from a server (supports many protocols).
    ```bash
    curl -I [https://example.com](https://example.com)
    ```
* **`ip`**: Modern tool for managing network interfaces and routing.
    ```bash
    ip addr show
    ```
* **`netstat`**: Displays network connections, routing tables, and interface statistics.
    ```bash
    netstat -tuln
    ```
* **`traceroute`**: Displays the path (hops) packets take to reach a host.
    ```bash
    traceroute google.com
    ```
* **`nslookup` / `dig`**: Queries DNS name servers for domain information.
    ```bash
    dig example.com
    ```
* **`scp`**: Securely copies files between hosts over SSH.
    ```bash
    scp file.txt user@remote:/path/
    ```
* **`rsync`**: Fast, versatile file copying tool that synchronizes files between locations.
    ```bash
    rsync -avz local_folder/ remote_user@remote_host:/remote_folder/
    ```

## 9. Utilities & Package Management
* **`history`**: Displays the command history list.
    ```bash
    history | grep "ssh"
    ```
* **`man`**: Displays the manual page for a specific command.
    ```bash
    man grep
    ```
* **`alias` / `unalias`**: Creates or removes shortcuts for long commands.
    ```bash
    alias ll='ls -lah'
    ```
* **`cal`**: Displays a simple calendar in the terminal.
    ```bash
    cal
    ```
* **`apt` / `dnf`**: Package managers for Debian-based (apt) or RHEL-based (dnf) systems.
    ```bash
    sudo apt install git
    ```
